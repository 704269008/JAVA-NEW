1.背景介绍

线程，有时被称为轻量级进程。 线程是进程中的一个实体，是被系统独立调度和分派的基本单位。 单个程序中同时运行多个线程完成不同的工作，称为多线程

Runnable and Thread

Thread实现了Runnable，至于两者的真正区别最主要的就是一个是继承，一个是实现；

使用Runnable可以避免java中的单继承的限制

多线程与多进程

1。单进程单线程：一个人在一个桌子上吃菜。

2。单进程多线程：多个人在同一个桌子上一起吃菜。

3。多进程单线程：多个人每个人在自己的桌子上吃菜。

锁与同步

当两个或者多个线程同时想要去获取共享资源的锁时, 但每个线程都要等其他线程把他们各自的锁给释放,才能继续运行,这就是死锁.
